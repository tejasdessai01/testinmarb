# Installation

Clone and navigate to the repository root.

Run the following commands

```bash
# Define environment variables
echo "PORT=8099" > .env
echo "NEXT_PUBLIC_API_BASE_URL=http://localhost:3099" >> .env

# Install dependencies and initialise the database
npm install
```

<br/>

# Development

```bash
npm run dev
```
